# svelte-breadcrumb

A dynamic breadcrumb svelte component

## Installation

```
npm install svelte-breadcrumb --save
```

## Usage

```javascript
import Breadcrumb from "svelte-breadcrumb";

<Breadcrumb stepsCount={10} activeStep={5}/>
```

#### PROPS (property - defaultvalue)
```javascript
stepsCount = 5;
activeStep = 1;
activeColor = '#0066f5';
defaultColor = '#999';
circleSize = 12;
linewidth = 30;
lineStrokeWidth = 2;
```
